# IM

IM 是一个基于 UDP socket + 多线程实现的聊天室(类似于 QQ 聊天，并且支持语音聊天)

### 基本功能

- IM 支持 QQ 中的基本功能

- 支持语音聊天

- IM 支持单人聊天

- IM 支持多人聊天，即群聊

- 支持消息回执，消息不丢失，保证数据传输的可靠性

- IM 支持用户状态信息推送，在线 or 离开

- 支持 IM 服务的开启、关闭及重启(通过命令控制 ./ctl.sh -s ···)
